## 동방 - 대학 동아리 교류 플랫폼

### 🔍 어떤 서비스 인가요?

동방은 대학 동아리들이 활동을 공유하고, 다른 동아리와 자연스럽게 교류할 수 있도록 돕는 커뮤니티 플랫폼입니다.

### ⚙️ 어떤 기술을 사용했나요?

`TypeScript` `Next.js` `Tailwind CSS`  
`Axios` `TanStack Query` `Supabase`  
`React Hook Form` `Yup`
